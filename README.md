A. Ringkasan Praktikum
- Pada pertemuan pertama ini, fokus utama praktikum adalah melakukan persiapan lingkungan pengembangan (development environment) untuk pengembangan aplikasi menggunakan Flutter.
Berikut adalah tahapan yang telah saya lakukan:

1. Instalasi Flutter SDK
Saya telah mengunduh dan menginstal Flutter SDK, kemudian mengatur environment variables agar perintah flutter dapat dijalankan melalui Command Prompt (CMD).

2. Pengecekan Kesehatan Sistem (Flutter Doctor)
Saya menjalankan perintah:
- flutter doctor
untuk memastikan seluruh komponen yang dibutuhkan sudah terpasang dengan benar.

3. Instalasi Android Studio dan Komponen SDK
Menginstal Android Studio beserta Android SDK dan Command-line Tools yang dibutuhkan untuk proses build dan pengujian aplikasi.

4. Persetujuan Lisensi Android
Menjalankan perintah berikut untuk menyetujui lisensi:
- flutter doctor --android-licenses
dan menyetujui seluruh lisensi yang diminta.

5. Pembuatan Project Pertama (Hello World)
Membuat project Flutter pertama dan menjalankan file hello_word.dart sebagai percobaan awal aplikasi.
